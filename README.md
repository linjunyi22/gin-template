# gin template

## 运行

* 下载

```bash
go get -u github.com/linjunyi22/gin-template
```

* 执行生成命令

  > gin-template new [project name]

```bash
gin-template new hello-world
```

* 生成简洁的 gin 项目框架，目录：
```
├── README.md
├── conf
│   └── app.conf
├── controllers
│   └── controller.go
├── go.mod
├── main.go
├── models
│   └── model.go
├── routers
│   └── router.go
└── utils
```
